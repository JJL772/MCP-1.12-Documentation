int net.minecraft.command.ICommandManager.executeCommand ( ICommandSender  sender,
String  rawCommand

Attempt to execute a command. This method should return the number of times that the command was executed. If the command does not exist or if the player does not have permission, 0 will be returned. A number greater than 1 can be returned if a player selector is used.
